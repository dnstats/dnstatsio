Glossary
========

.. glossary::
   DMARC
      DMARC or Domain-based Message Authentication, Reporting and Conformance

   CAA
      DNS Certification Authority Authorization

   SPF
      SPF or Sender Policy Framework is method to help detect fraudulent emails. See :rfc:`7208`.

   Certificate Authority
      Certificate Authority or CA

   DNSSEC
      DNS SEC

   DNS
      Domain Name Service
